## ğŸ”§ SERVICE DETAILS

### **gbpAuthService.ts** (300 lines)
OAuth 2.0 authentication for Google Business Profile.
- `getAuthorizationUrl()` - Generate login URL
- `exchangeCodeForToken()` - Convert auth code to token
- `getValidAccessToken()` - Get valid token with auto-refresh
- `refreshAccessToken()` - Renew expired tokens
- `isAuthenticated()` - Check auth status
- `logout()` - Clear credentials

### **gbpDataService.ts** (350 lines)
Fetch GBP data and manage posts/reviews.
- `getBusinessProfile()` - Get profile info
- `getInsights()` - Fetch metrics
- `getPosts()` - List posts
- `createPost()` - Publish post
- `getReviews()` - Fetch reviews
- `replyToReview()` - Auto-respond

### **postAutomationService.ts** (380 lines)
Schedule and automate daily posts.
- `createScheduleTemplate()` - Create schedule
- `getScheduleTemplates()` - List schedules
- `updateScheduleTemplate()` - Modify schedule
- `deleteScheduleTemplate()` - Remove schedule
- `generateAndPublishPost()` - AI generation & publish
- `schedulePost()` - Queue for future

### **socialMediaService.ts** (320 lines)
Manage accounts across 4 social platforms.
- `connectAccount()` - Add platform account
- `disconnectAccount()` - Remove account
- `getConnectedAccounts()` - List accounts
- `postToSocialMedia()` - Publish post
- `crossPost()` - Multi-platform publish
- `getOAuthUrl()` - OAuth authorization

### **websiteAnalysisService.ts** (420 lines)
Analyze websites for SEO and performance.
- `analyzeWebsite()` - Run full audit
- `getPerformanceMetrics()` - Performance data
- `checkSEO()` - SEO audit
- `checkAccessibility()` - Accessibility check
- `checkSecurity()` - Security scan
- `getOptimizationRecommendations()` - Suggestions
